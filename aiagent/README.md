# AI Agent Project

This repository contains various front-end and back-end implementations for a chat application powered by the Qwen language model.

## Backend

*   **Directory**: `aiagent_backend`
*   **Description**: A Python Flask server that acts as a bridge between the front-end clients and the Ollama-hosted Qwen model.
*   **Model**: The backend currently uses the `qwen:0.5b` model running via Ollama.

## Front-ends

### Web Interface

*   **Directory**: `aiagent_front`
*   **Description**: A simple, browser-based chat interface built with HTML, CSS, and JavaScript.

### Python macOS App (Briefcase)

*   **Directory**: `aiagent_pymac`
*   **Description**: A native macOS application built with Python and the Toga UI toolkit, created using the Briefcase tool.

### Objective-C macOS App

*   **Directory**: `aiagent_objc`
*   **Description**: A native macOS application built with Objective-C and AppKit.
*   **Status**: **Warning: This application is currently not functional and has known issues.**
